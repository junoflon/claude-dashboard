설정 가이드
1
Discord 봇 생성
Discord Developer Portal → New Application → Bot 탭 → Reset Token → 복사
2
봇 권한 설정
Bot 탭 → Privileged Gateway Intents → Message Content Intent 켜기
3
디스코드 서버에 채널 만들기
#설계, #디버그, #리서치, #배포, #리뷰 채널 5개 생성
4
봇 초대
OAuth2 → URL Generator → bot + applications.commands → Send Messages, Read Messages → URL로 초대
5
.env 파일 설정
DISCORD_TOKEN=봇토큰 + DISCORD_CHANNEL_ID=알림채널ID
6
서버 재시작
node server.js — 봇이 자동으로 채널을 감지하고 전문가를 매핑합니다
